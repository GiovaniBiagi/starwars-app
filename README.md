# Star Wars Explorer

## Tech Stack

- React + TypeScript
- React Router v7
- TailwindCSS
- Zod
- Vite

## Prerequisites

- Node.js 20.x
- npm or yarn
- Git

## Installation

```bash
# Clone repository
git clone <repository-url>

# Install dependencies
npm install

# Create .env file
echo "VITE_API_URL=https://swapi.dev/api" > .env

# Start development server
npm run dev
```
